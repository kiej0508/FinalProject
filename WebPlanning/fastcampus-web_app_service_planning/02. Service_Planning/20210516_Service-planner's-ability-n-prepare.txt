- 서비스 기획자의 역량 및 준비 -

[ 28. 구직 공고를 읽는 방법 ]
	1. 구직공고를 통해 역량 보기
	
	2. 기획자에게 필요한 역량 정리
		1) 사업 기획 능력 : 비즈니스 분석, 시장 흐름 이해
		2) 서비스 기획 능력 : 고객 분석, 고객 경험, 플랫폼 생태계 선순환 기획
		3) 데이터 분석 능력 : 행동 분석(GA 등), 사용성 테스트, 린 UX 실행
		4) 프로젝트 방법로녈 실무 능력 : 폭포수 / 애자일 각 산출물의 작성
		+ 추가 능력
			- 디자인, 개발 업무 대행 능력 : 포토샵, 스케치, 프로토타이핑, 퍼블리싱 일부
			- 전략 기획 및 IR 자료 등 문서 작업 능력
	
	3. 기획자에 필요한 태도
		1) 개발, 디자인툴을 익히는 것은 장기적 성장에는 도움이 된다
		2) 커뮤니케이션, 자사의 용어사전을 만들어서 어휘를 통일시켜야 한다	

[ 29. 포트폴리오의 준비 ]
	1. 기획자 포트폴리오란?
		- 자신의 경력이나 실력을 알 수 있도록 과거의 작품이나 관련 내용을 모은 묶음
		- 기획자다운 역량을 보여줄 수 있도록 구성
		- 비즈니스와 고객에 대한 이해와 문제해결을 위한 사고 과정, 산출물 작성 능력을 보여줄 수 있어야 함
	
	2. 기획자 포트폴리오 구성
		1) 서비스 기획 강조
			- 서비스 화면보다 특정 시장의 Player들의 특징, 수익구조, 비즈니스 모델의 분석
			- 추가적으로 제안하고 싶은 서비스에 대해서 강조
			※ 제안서 형식의 포트폴리오를 만들어본다!
		2) UX 분석 강조
			- 특정 기능이나 Task에 대하여 여러 서비스의 고객 UX를 분석하고 이용 동선과 장단점 분석
			- 인사이트 정리 및 해당 Task에 대한 자신의 이상적인 서비스 형태 제안
		3) 실무 능력 강조
			- 간단한 프로젝트 진행해보고 작업했던 문서 보여주기
			- 기존 서비스 분석 후 '데이터'와 '로직' 등 개발 구현의 관점에서 디스크립션이 상세화된 화면설계서를 역기획
			※ 역기획 : 이미 있는 서비스에 대해서 스토리보드를 반대로 만들어보는 것 / 정확히는 모르더라도 이렇게 동작하는구나를 만들어본다

[ 30. 서비스 기획자만의 스타일과 아이덴티티를 만들어라 ]
	1. 기획자에게 커뮤니케이션이 중요하다
		- 끝까지 듣고 상대방의 입장을 이해하는 태도
		- 두려움을 갖기 보다는 문제 상황에 대해서 발견하고 함꼐 해결하려는 태도
		- 모르는 것을 부끄러워하지 않고 물어보는 태도
	
	2. 기획자 커뮤니케이션 스타일
		1) 친분 강조형
			- 협업 대상자와 사적인 관계를 잘 형성하여 편현 관계를 통해 이슈와 감정을 해결하는 타입
			- 외향적인 성격이며 인간적 매력이 많은 스타일
			- 단점 : 강제력
		2) 조곤조곤 설득형
			- 협업 대상자와 필요 이상의 관계를 맺기 보다는 요건과 문서를 확실하게 정리하여 대화하는 유형
			- 내향적, 냉소적 성격이나 결단력 있는 스타일
			- 단점 : 어려워함
		3) 하이브리드형
			- 리뷰하거나 사적인 이야기할 때와 중요한 대화나 오류 등에 대해서는 단호하게 대처하는 유형
			- 반대의 하이브리드 유형도 가능

[ 31. 서비스 기획자의 미래 - 신기술과 서비스 기획 ]
	1. 서비스 기획 ≠ 모바일 웹/앱을 만드는 것
		- 서비스 기획이란 디지털적 속성을 가진 서비스를 초기부터 구상하여 실제 개발 가능한 수준에서 구현할 수 있도록 프로덕트를 정의하고 구현 프로젝트를 이끌어 가는 사람
		- 디지털 서비스를 책임지는 서비스의 기본 개발 구조는 모두가 비슷
		→ 디지털 서비스의 영역이 넓어질수록 기획자가 할 수 있는 서비스의 범주도 넓어짐
	
	2. 눈여겨봐야할 트렌드
		1) AI 서비스의 확대
			- AI는 데이터를 기반으로 학습되는 '로직 구현'의 산물
			- 단, 서비스 목표가 없다면 학습 방향성을 잡거나 로직을 설정할 수가 없다
			- 애초에 특정 활용 서비스 목표를 가진 데이터 수집이 필요
		2) 블록체인 서비스
			- 단순 코인이라는 가상 화폐가 아니라 분산화 컴퓨팅 플랫폼으로서의 블록체인
			- 코인 생태계를 이해한 뒤 이에 적합한 서비스 기획이 필요
		3) 디지털 트랜스포메이션
			- 오프라인 유통매장, 생산공장 등 모든 오프라인 매장이 디지털화되어 데이터가 축적되고 자동화되는 현상
			- 네트워크와 디지털적 속성이 있는 곳은 서비스 기획의 영역
